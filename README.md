# Statistics in Python

This repository contains a collection of Jupyter Notebooks (`.ipynb`) and Python scripts developed as part of a **Statistics in Python** course. The project explores fundamental statistical concepts, data analysis techniques, and machine learning implementations using Python's scientific computing stack.

## Repository Contents

Below is a breakdown of the topics covered in this repository:

| File Name | Description |
| :--- | :--- |
| `Kernel_Density_Estimation.ipynb` | Implementation and visualization of KDE to estimate the probability density function of random variables. |
| `Histogram_&_PDF.ipynb` | Exploration of data distributions using histograms and Probability Density Functions (PDFs). |
| `Feature_Selection_Using_PDF.ipynb` | Application of statistical methods (PDF-based) for feature selection in machine learning pipelines. |
| `Bayesian_Inference_for_Multiclass.ipynb` | Practical application of Bayesian inference techniques for classification problems. |

## Technical Stack
* **Language:** Python 3.x
* **Key Libraries:** `numpy`, `pandas`, `matplotlib`, `scipy`, `scikit-learn`

## Getting Started
To run these notebooks locally, ensure you have a Python environment set up with the necessary dependencies:

```bash
# Clone the repository
git clone <your-repository-url>

# Install required packages
pip install numpy pandas matplotlib scipy scikit-learn jupyter
```

Then, launch Jupyter Notebook:
```bash
jupyter notebook
```
